# Interactive PID Tuner

A **real-time, interactive web application** to demonstrate PID control. Users can adjust **Kp, Ki, Kd** sliders and see the effect on a simulated system, like a **self-balancing robot** or **rotating arm**. Includes live animation, optional graphs, disturbances, and tooltips explaining PID behavior. Built with **HTML, CSS, and JavaScript**.  

---

## Features

- Adjust **Kp, Ki, Kd** in real time.  
- Animated simulation of system response.  
- Optional graph: **target vs. actual position**.  
- Disturbance/noise toggle to see PID handling.  
- Reset button to restart simulation.  
- Responsive design for desktops and tablets.  

---

## How to Run Locally

1. **Clone the repository**:  
```bash
git clone https://github.com/your-username/interactive-pid-tuner.git
cd interactive-pid-tuner
```

2. **Open `index.html`** in your browser (double-click or via a live server in VSCode).  

3. **Adjust sliders** for Kp, Ki, Kd and watch the simulation update in real time.

## Hosting Online with GitHub Pages

1. Go to your repository on GitHub.  
2. Click **Settings → Pages**.  
3. Under **Build and deployment**, select **Deploy from a branch**.  
4. Choose the **main** branch and `/root` folder.  
5. Click **Save**.  

GitHub will provide a link like:  
```
https://your-username.github.io/interactive-pid-tuner/
```


Share this link to give anyone access to your live PID demo.
